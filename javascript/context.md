# 执行上下文

## 定义
也称为作用域。为了查找变量。

作用域也可以根据代码层次分层，以便子作用域可以访问父作用域，通常是指沿着链式的作用域链查找，而不能从父作用域引用子作用域中的变量和引用。

都包含什么？
变量对象、作用域链、this 指向 

## 执行上下文的类型
|  作用域（Scope）   | 范围  |
|  ----  | ----  |
| window/global Scope  | 全局作用域 |
| function Scope  | 函数作用域 |
| Block Scope  | 块作用域（ES6） |
